# Java Kargo Takip Sistemi

Java programlama dili ile geliştirilmiş, .txt dosyları üzerinden çalışan kargo takip sistemi.

 * Paket ekleme/silme
 * Araç ekleme/silme
 * Depo paket listeleme
 * Araça paket yükleme
 * Araçta yüklü paketleri depoaya indirme
